# Visual Data Prep

`Visual Data Prep` は、CSV / Excel / JSON などの表データを、ノードベースの画面で視覚的に整形・結合・集計できるデスクトップツールです。

プログラミングや SQL に不慣れでも、

- ファイルを読み込む
- ノードをつなぐ
- 条件や変換を設定する
- 結果を確認して書き出す

という流れで、データ前処理を素早く組み立てられます。

## 主な用途

- 複数 CSV / Excel の結合
- 売上データや顧客データの整形
- 月次レポート作成前の前処理
- 名寄せや列加工などのクレンジング
- グラフ表示前の集計データ作成
- SQL 化したい処理フローの可視化

## 主な特徴

- ノーコード操作
  ノードを配置して線でつなぐだけで、処理手順を視覚的に構築できます。
- ローカルファイル対応
  `CSV` `XLSX` `JSON` の読み込みに対応しています。
- Auto Folder 対応
  指定フォルダ内の最新ファイルを自動で読み込めます。
- Paste Data 対応
  表エディター上で、直接セル編集・一括貼り付け・範囲指定ができます。
- SQL 相互変換
  作成したフローから `SELECT` 文を生成でき、SQL からノード構成を作ることもできます。
- プレビューとダッシュボード
  処理結果をテーブルとチャートで確認できます。
- 大規模データ向け最適化
  プレビュー行数制御、自動停止、計算キャッシュ、Web Worker による処理分離を入れています。
- 操作補助
  右クリックからのノード追加、ツールチップ、カメラフォーカス設定、AutoConnect を備えています。

## 利用できる主なノード

- `Source`
  ローカルの `CSV` `Excel` `JSON` ファイルを読み込みます。
- `Auto Folder`
  指定フォルダ内の最新ファイルを自動で読み込みます。
- `Paste Data`
  貼り付けたデータを表形式で編集し、そのまま入力ソースとして使えます。
- `Union`
  2つのデータを縦に結合します。
- `Join`
  共通キーでデータを横結合します。
- `VLOOKUP`
  別データを検索して列を追加します。
- `Minus`
  一方に存在するデータを除外します。
- `Transform`
  文字列加工、ゼロ埋め、置換、条件分岐、オートナンバー付与などを行います。
- `Calculate`
  2列を使った計算や結合を行います。
- `Select`
  必要な列だけを残します。
- `Filter`
  条件に合う行だけを抽出します。
- `Sort`
  指定列で昇順・降順に並び替えます。
- `Group By`
  集計処理を行います。
- `JSON Array`
  `["json","sample","demo"]` のような JSON 配列文字列を展開し、1要素1行のテーブルへ変換します。
- `Visualizer`
  棒グラフや折れ線グラフで可視化します。

## できること

- 複数ファイルの結合と比較
- 必要列の抽出と不要列の削除
- 条件に応じた値の変換
- 新規列追加と列位置指定
- オートナンバー列の作成
- JSON 配列文字列の展開
- テーブル結果のプレビュー
- チャート表示
- `CSV` `XLSX` `JSON` へのエクスポート
- フローの保存と再読み込み

## 使い方

1. `Source` または `Paste Data` ノードでデータを用意します。
2. 必要なノードをつないで処理フローを作ります。
3. 各ノードで条件や対象列を設定します。
4. 下部プレビューで結果を確認します。
5. 必要に応じて `CSV` `XLSX` `JSON` へ出力します。

## 便利な操作

- 左側 `Toolbox` からノードをドラッグ＆ドロップできます。
- 何もないキャンバス上で右クリックすると、ノード追加パレットを表示できます。
- 新規ノード追加時は、直前ノードへの自動接続を `AutoConnect` で切り替えられます。
- ノード接続、追加、削除、リサイズ時のカメラフォーカスは個別に ON / OFF を設定できます。
- ノードをドラッグ中のみゴミ箱が表示され、ドロップで削除できます。
- ノード左上のボタン、またはヘッダーのダブルクリックで最小化できます。
- ノード右上の `?` からヘルプを確認できます。
- プレビューは表示件数を変更でき、重いデータは自動停止します。

## 保存される設定

次の設定は再起動後も引き継がれます。

- テーマ
- ツールチップ表示
- カメラフォーカス設定
- Toolbox の折りたたみ状態
- 下部データテーブルの高さ
- プレビュー表示件数
- AutoConnect 設定

## 動作環境

- Windows での利用を想定した Electron アプリです
- `Node.js` と `npm` があれば開発環境から起動できます
