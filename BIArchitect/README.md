
＊ 概要

  BI Architectは、
  ドラッグ＆ドロップで直感的にデータ処理パイプラインを作れる、ブラウザやデスクトップ向けのノーコードツールです。
  
  ExcelやCSV、Web APIからのデータ読み込みから、
  データの結合、クレンジング、集計、グラフ化までのすべてのステップを、
  ノードをつなぐだけで簡単に実行できます。
  
  作ったフローをSQLに変換したり、SQLからフローを自動作成したりする機能も備えています。


＊ 主な機能

  ・ノーコードでパイプライン構築: 
  React Flowを使った直感的な操作画面で、プログラミングの知識がなくても誰でも簡単に扱えます。
  
  ・多彩なデータソースに対応: 
  パソコン内のファイル読み込みだけでなく、指定したフォルダ内の「最新ファイル」を自動で読み込む機能や、URLからWebデータを取得する機能に対応しています。
  
  ・強力なデータ変換 (Transform): 
  データの結合や絞り込みに加えて、特定の条件のときだけ文字を抽出したり、ゼロ埋めや四捨五入などの細かなデータ整形が可能です。
  
  ・フローとSQLの相互変換: 
  作成したノードの構成からSQLを自動生成できます。逆に、SQLを入力してノードを自動で配置することも可能です。
  
  ・グラフとダッシュボード: 
  処理したデータはすぐに表形式で確認でき、棒グラフや折れ線グラフを作ってダッシュボードとして一覧表示できます。
  
  ・データのエクスポート: 
  処理結果をCSVやExcelファイルとして保存したり、ダッシュボードをPDFとして出力したりできます。プロジェクト自体の保存や読み込みも可能です。


＊ 利用可能なノード一覧

  パソコン内のCSVやExcelファイルを読み込みます。読み込む範囲やヘッダーの設定も可能です。
  
  ・Auto Folder：指定したフォルダを見張って、一番新しいファイルを自動的に読み込みます。
  ・Web Source：URLを入力するだけで、JSON、CSV、Webページ内の表データを自動で取得します。
  ・Union：2つのデータを「縦」につなぎ合わせて1つにします。
  ・Join：2つのデータを共通のキーで「横」にくっつけます。
  ・VLOOKUP：別のデータからキーを検索し、特定の列の値を新しい列として追加します。
  ・Minus：メインのデータから、サブのデータにあるレコードを差し引きます。
  ・Group By：データをグループごとにまとめて、合計や件数を計算します。
  ・Sort：指定した列のデータを昇順（小さい順）や降順（大きい順）に並び替えます。
  ・Transform：文字列の操作、数値の丸め、条件ごとの処理など、データの整形を細かく行います。
  ・Select：必要な列だけを選んで残し、いらない列を削除します。
  ・Filter：「〇〇以上」「〇〇を含む」など、条件に合うデータだけを絞り込みます。
  ・Visualizer：グラフのX軸とY軸を指定して、データを視覚的に表示します。

＊ 使い方と便利な操作 (Tips)

  ・ノードの配置: 左側のツールボックスから、使いたいノードをキャンバスにドラッグ＆ドロップしてください。
  ・線の削除: つないだ線を消したい時は、線を右クリックするだけで削除できます。
  ・ノードの最小化: ノードの左上にある「▲/▼」ボタンを押すか、ノードの上の部分をダブルクリックすると、小さく折りたためて画面をすっきりさせることができます。
  ・自動カメラフォーカス: 画面の上の「Camera Focus」をオンにすると、ノードを操作した時に画面の中央へゆっくりとカメラがついてきます（画面酔いしにくいように、優しい速度に調整されています）。
  ・ヘルプ機能: 各ノードの名前の横にある「?」マークをクリックすると、詳しい使い方やヒントが表示されます。


＊ 開発環境とインストール方法

  このアプリは React、Vite、Electron を使って作られています。  

  -- セットアップ

    # 必要なパッケージをインストールします
    npm install
    
    # 開発モードで起動してブラウザで確認します
    npm run dev

  -- アプリケーションのビルド（実行ファイルの作成）

    # Windows用の実行ファイル（.exe）を作成します
    npm run build:win
    
    # Mac用の実行ファイル（.dmg）を作成します
    npm run build:mac
    
    ビルドが終わると、dist フォルダの中にインストーラーが作成されます。

＊ 使用している技術

  ・フロントエンド: React, TypeScript, Tailwind CSS
  ・フローエディタ: React Flow (@xyflow/react)
  ・データ処理: SheetJS (xlsx), PapaParse
  ・グラフ描画: Recharts
  ・アイコン: インラインSVG (Lucideのデザインを参考にしています)
